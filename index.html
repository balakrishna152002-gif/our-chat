🔥 Firebase Setup

This takes ~5 minutes and is completely free. Firebase lets your messages sync in real-time between devices.

1
Go to console.firebase.google.com → Create a project (any name)
2
Click Realtime Database → Create database → Start in test mode
3
Go to Project Settings → Add a web app → Copy your config values below
